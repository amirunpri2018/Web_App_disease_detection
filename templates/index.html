Blood Smear Analysis(BSA) for Parasitized Cell detection on Malaria dataset of NIH using Convolutional Neural Network with 98.21% Accuracy


Instruction to use: Click on select Image , Select the Microcoscopic image which needs to be tested, click Predict ,Then It will run the model in background and will give you the result

In the proposed scheme twelve hidden layers are used in the CNN that can be listed as-

 Convolution layer 1 : 3 x 3 filters, 3 channels and 32 filters

 Convolution layer 2 and 3 : 3x3 filters, 32 channels and 32 filters

 Pooling layer : max pooling of window size 2 x 2 and stride 2

 Fully connected layer with 1024 hidden units
